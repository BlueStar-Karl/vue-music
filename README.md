# vue-music

> Vue.js 音乐 App


## 项目主要技术栈

vue2.0 vuex node 


### 安装与运行

```
git clone https://github.com/songhaoreact/vue-music.git

cd vue-music

npm install

npm run dev //服务端运行 访问 http://localhost:8080

npm run build 项目打包 



```
